ðŸš€ Deployment Instructions

To deploy all 5 projects on Vercel:

1

Create vercel.json

Add this file to your repository root:

{
  "routes": [
    {
      "src": "/shopping-cart-css/(.*)",
      "dest": "/Experiment-01_Shopping-cart-app_CSS/Shopping-cart-app/public/html/$1"
    },
    {
      "src": "/shopping-cart-bootstrap/(.*)",
      "dest": "/Experiment-02_Shopping-cart-app_Bootstrap/ShoppingCartApp/public/html/$1"
    },
    {
      "src": "/weather-vue/(.*)",
      "dest": "/Experiment-04_Weather_information_Visualization_ES6/Weather-info-app/public/html/$1"
    },
    {
      "src": "/weather-react/(.*)",
      "dest": "/Experiment-12_ReactJS_Weather-app/weather-app/weather-app/public/$1"
    },
    {
      "src": "/todo-react/(.*)",
      "dest": "/Experiment-13_ReactJS_Todo-app/todo-app/public/$1"
    }
  ]
}
2

Create package.json

Add this file to your repository root:

{
  "name": "portfolio",
  "version": "1.0.0",
  "scripts": {
    "build": "echo 'Static projects require no build'"
  }
}
3

Deploy to Vercel

Push to GitHub and import to Vercel:

  1. Create account at vercel.com
  2. Click "Add New Project"
  3. Import your GitHub repository
  4. Set framework to "Other"
  5. Click "Deploy"
